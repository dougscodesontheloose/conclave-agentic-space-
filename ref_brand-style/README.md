# Brand Style References

This directory stores brand voice and writing style analysis documents.

## How to Use

Place your brand's writing samples, tone analyses, and style guides here.
The content squad agents will reference these files to match your writing voice.

## Suggested Contents

- `writing-tone-analysis.md` — Analysis of your brand's writing tone and patterns
- `vocabulary-guide.md` — Preferred terms, forbidden phrases, and jargon rules
- `content-examples/` — Folder with your best content pieces as writing references
