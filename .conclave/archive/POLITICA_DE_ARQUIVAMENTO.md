# Diretrizes de Proteção de Dados (Conclave Archive)

Nenhum arquivo deste projeto deve ser DELETADO de forma permanente por um agente autônomo. 

## Procedimento de Movimentação:
1. Antes de substituir ou remover um arquivo, ele deve ser movido para esta pasta.
2. O nome deve conter o timestamp ISO para evitar colisões.
3. Se houver dúvida sobre a utilidade do arquivo, ARQUIVE. Não remova.

Esta pasta atua como o 'Fail-Safe' contra alucinações de modelos ou instruções mal interpretadas.
