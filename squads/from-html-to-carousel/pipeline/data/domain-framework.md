# Framework de Operação: From HTML to Carousel

### Metodologia de Transformação
1. **Extração de Sentido**: Ler o texto denso ou artigo original e mapear a tese principal (hook).
2. **Estruturação Funcional**: Quebrar o texto em Tópicos Frasais, cada um correspondendo a um Slide.
3. **Lapidação de Copy**: Reduzir as frases para limites de 25 palavras por slide, eliminando advérbios e adjetivos desnecessários.
4. **Criação da Legenda**: Acompanhar o arquivo do carrossel com um post adequado, gerando o gancho para a leitura do PDF.
5. **Adaptação Visual**: Enviar o copy para o Designer, para que seja aplicado no construtor de HTML, garantindo as cores da marca e contraste.
6. **Auditoria de Qualidade**: Validar visual e copy simultaneamente antes da exportação.
