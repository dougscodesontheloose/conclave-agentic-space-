# Antípadrões

### Never Do
1. **Paredes de texto na imagem**: O usuário de rede social não para para ler blocos de texto maiores que 3 linhas finas em uma imagem. O alcance cai se a rejeição inicial aumenta.
2. **Excesso de termos técnicos confusos**: O objetivo é explicar de modo estratégico, não alienar o público. Jargão pelo jargão demonstra insegurança em vez de autoridade.
3. **Múltiplos CTAs**: Pedir para o usuário "curtir, comentar sua opinião, salvar pra depois e clicar no link da bio" no slide final paralisa a tomada de decisão.
4. **Infantilizar conceitos difíceis (Dumbing down)**: Evite forçar analogias fofas para executivos (ex: comparar um pipeline de dados a uma montanha russa infantil).

### Always Do
1. **Frases de efeito ancoradas em dados**: Se fizer uma alegação forte ("Esqueça as campanhas de last click"), justifique no slide seguinte.
2. **Listas numeradas curtas**: A escaneabilidade sobe em 60% se as pessoas perceberem o progresso (ex: "Existem 3 erros...", enumerar erro 1, erro 2...).
3. **Construção de tese**: Cada slide deve interligar o contexto do anterior, montando uma escadinha lógica.
