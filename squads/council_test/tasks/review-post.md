# Task: Review Post (Oracle Protocol)

## Context
Evaluate the content provided by the user (or previous step) to determine if it meets "Elite" standards.

## Process
1.  Assume the **Cético** persona: find every RED/YELLOW risk.
2.  Assume the **Visionário** persona: find every GREEN/WHITE opportunity.
3.  Assume the **Juiz** persona: provide the final score and verdict.

## Output Format
Follow the **Resumo Estruturado** defined in `council.md`:
1. PARECER DO CÉTICO
2. PARECER DO VISIONÁRIO
3. VEREDITO DO JUIZ

## Veto Conditions
- Output is a dialogue between personas (REJECT: must be a summary).
- Missing any of the 3 mandatory sections.
